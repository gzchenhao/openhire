## A. GitHub

### A1 · 注册 GitHub 账号
网页 `https://github.com/signup` → 输入邮箱、密码、用户名 → 按邮件验证码完成。（只有你能做。）

### A2 · 新建仓库 `openhire`
右上角 **+** → **New repository** → Repository name 填 `openhire` → 选 **Public** → **不要**勾 "Add a README"（我们本地已有）→ 点 **Create repository**。（也 **【可代劳】**：你先在本机跑一次 `! gh auth login` 登录，我就能用 `gh repo create` 替你建。）

### A3 · 把本地代码推上去　**【可代劳】**
本机 `C:\openhire` 目前还不是 git 仓库。这步我可代劳：我来 `git init` → 提交（`.gitignore` 已确保 `.env`/`.pypirc`/`*.db`/`dist` 不进库）→ 加你的远程地址 → 推送。你只需先完成 A2 并让我拿到推送权限（`gh auth login` 或给我一个仓库范围的 GitHub token，走本机环境变量）。
> 你若想自己做：在 `C:\openhire` 依次 `git init`、`git add .`、`git commit -m "OpenHire v0.1"`、`git branch -M main`、`git remote add origin https://github.com/你的用户名/openhire.git`、`git push -u origin main`。

### A5 · 生成并上传快照附件　**【可代劳】**

> 传完后验证：Release 里该附件的下载地址应等于 `src/openhire/config.py` 的 `OPENHIRE_SNAPSHOT_URL`（我在第 0 步 wire 用户名时会对齐）。

## B. PyPI

### B1 · 注册 PyPI 账号
网页 `https://pypi.org/account/register/` → 邮箱 + 用户名 + 密码 → 收邮件点验证链接。（只有你能做。）

### B2 · 开启 2FA（PyPI 强制）
登录后网页 `https://pypi.org/manage/account/` → 找 **Two-factor authentication (2FA)** → 用手机上的验证器 App（如 Google Authenticator/Microsoft Authenticator）扫二维码 → 输入 6 位码确认 → **把恢复码抄下来存好**（丢了会锁号）。（只有你能做。）

### B3 · 生成 API token
同一页面往下找 **API tokens** → **Add API token** → Token name 填 `openhire-upload` → Scope 首次选 **"Entire account"**（因为包还没上传过，无法按项目限定；上传成功后可删掉它、改发项目范围的新 token）→ 点 **Create** → 页面会显示一串 `pypi-AgEI...` 开头的 token，**只显示这一次**。（只有你能做。）

### B4 · 把 token 安全交给我　**【可代劳上传】**
把那串 `pypi-...` token 发我。我会**只**把它放进本机环境变量（`$env:TWINE_PASSWORD`，用户名用 `__token__`）或本机 `%USERPROFILE%\.pypirc`——**绝不写进任何代码、绝不 commit**（`.gitignore` 已挡 `.pypirc`）。
> 红线复述：token 只活在你这台电脑的环境变量/`.pypirc` 里；仓库里永远看不到它。

### B5 · twine 上传　**【可代劳】**
我来跑：`python -m twine check dist/*`（校验元数据）→ `python -m twine upload dist/*`（上传 sdist+wheel）。成功后包页在 `https://pypi.org/project/openhire/`。
> 你若想自己做：`pip install twine` 后同样两条命令，提示输入用户名填 `__token__`、密码粘贴 token。

### B6 · 验证全网可装
在**另一台**或干净环境：`pipx install openhire` → `ohp version` 应回 `openhire 0.1.0` → `ohp bootstrap` 能拉到 A5 的快照。（**【可代劳】**：我在本机新建隔离环境从 PyPI 实测。）
